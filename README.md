# ☕ Give Me More Coffee

**"당신의 취향에 맞는 최고의 원두를 찾아드립니다."**

## 📖 프로젝트 개요 (Overview)

**Give Me More Coffee**는 모바일 환경에 최적화된 **커피 원두 전문 이커머스 플랫폼**입니다.

사용자가 다양한 원산지와 가공 방식의 스페셜티 원두를 쉽고 빠르게 탐색하고, 원하는 용량에 맞춰 구매할 수 있는 서비스를 목표로 합니다. 현재 최신 상품 조회, 베스트셀러 추천, 그리고 상세한 상품 옵션 선택 기능을 제공하며, 결제 및 배송 시스템으로 확장 중입니다.

## ✨ 주요 기능
- **홈 화면:** 최신 등록된 원두와 가장 인기 있는(Best) 원두를 한눈에 파악
- **상품 상세:** 원두의 원산지, 가공 방식, 테이스팅 노트 확인 및 이미지 뷰어
- **옵션 선택:** 사용자가 원하는 중량(200g, 500g 등)과 수량 커스터마이징
- **회원 서비스:** 로그인, 회원가입, ID/PW 찾기 UI 제공 (연동 예정)
